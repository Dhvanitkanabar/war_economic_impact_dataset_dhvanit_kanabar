# WarLens — Global War Economic Impact Analysis Platform

## Project Overview
WarLens is a full-stack analytics platform designed to analyze and visualize the economic impact of global wars and conflicts.

## Tech Stack
- **Backend**: Node.js, Express.js, MongoDB
- **Frontend**: React.js, Vite, Tailwind CSS, Recharts
