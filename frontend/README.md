# WarLens Frontend

## Overview
WarLens is a professional web application designed for economic impact analysis of conflicts. This frontend project is built as a high-performance single page application to provide researchers and analysts with interactive visualizations, dashboards, and data filters.

## Tech Stack
- **React**: Component-based UI library
- **Vite**: High-performance frontend build tool
- **JavaScript (ES6+)**: Core programming language
- **Redux Toolkit & React-Redux**: State management (initialized)
- **React Router DOM**: Client-side routing (initialized)
- **Axios**: HTTP client for API requests (initialized)
- **Lucide React**: Premium icon pack (initialized)

## Available Scripts
Inside the `frontend` directory, you can run the following commands:
- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the production bundle of the application.
- `npm run preview`: Previews the production build locally.
